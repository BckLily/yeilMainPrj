;========================
;레벨업 시 획득 가능한 스킬(perk)
;
;스테이터스 증가(최대 5번)
;공격력 1증가
;방어력 0.5 증가
;체력 10증가
;
;직업 별 특성(능력)(최대 3번)
;소총병: 최대 탄약 15% 증가, 공격 속도 10% 증가
;위생병: 회복량 15%증가, 회복 속도 10% 증가
;공병: 방어물자 건설 속도 15%증가, 방어물자 수리 속도 10% 증가.
;
;========================
;특전 구매 시 활성화되는 스킬(perk)
;직업별 특전
;
;소총병
;1단계: 공격속도 15%증가
;2단계: 25% 확률로 탄 소모 없음
;3단계: 20% 확률로 공격력 20증가.
;
;위생병
;1단계: 회복 아이템 회복량 15%증가.
;2단계: 회복 아이템 사용속도 30% 증가.
;3단계: 20% 확률로 회복 아이템 소모 없음.
;
;공병
;1단계: 건설 속도 15% 증가
;2단계: 건설한 방어물자 최대 체력 70증가
;3단계: 방어 물자의 체력 일정 시간마다 자동 회복
;
